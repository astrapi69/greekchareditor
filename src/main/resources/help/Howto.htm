How to use the application

The use of the application is kept simple. You write the desired text wich has to be converted in the Input-editor and push the Button "transform". If you push the button "Copy HTML-Entities to Clipboard" or "Copy ISO-8859-7 to Clipboard" the appropriate content to the clipboard. So you do not need to mark(ctrl+a) and copy(ctrl+c). With the Button "Clear" you clear(empty) the content of all editors.
