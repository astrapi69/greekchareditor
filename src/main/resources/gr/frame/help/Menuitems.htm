Menuitems

  • The "File-menu" has the menuitems:

    • New. With "New" you clear all editors so you can create a new file.
    • Open. With "Open" you can open an existing file.
    • Save. With "Save" you can save the content of all editors in a file.
    • Save as. With "Save as" is allmost the same like "Save" but you can specify another directory and name for the file.
    • Exit. With "Exit" you finish the programm.

  • The "Edit-menu" has the menuitems:

    • The "Look and Feel"-menu has the menuitems:
      • Metal. With the "Metal"-menuitem is the default gui(graphical user interface).
      • Motiv. With the "Motiv"-menuitem the look and feel is like you know it from emac.
      • Windows. With the "Windows"-menuitem the look and feel is like you know it from Windows.

  • The "Help-menu" has the menuitems:

    • Content: This is the helpcontent that you are just reading.
    • License: This shows the license from the application.
    • Info: With the "Info"-menuitem opens a Dialog with the information about the developer.
